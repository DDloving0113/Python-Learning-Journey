本模块为扁平的独立脚本集合，按主题分为三类：
- 标准库 csv 演示：`demo_fieldnames_rule.py`、`demo_auto_fieldnames.py`、`demo_dict_loop.py`、`demo_dict_values.py`、`demo_fieldnames.py` 围绕 `csv.DictReader`/`DictWriter` 的 fieldnames 规则与字典遍历；`ex32_csv_stats.py`、`ex33_csv_stats_export.py`、`ex34_department_top_earner.py`、`practice_csv_advanced.py` 用纯 Python 字典完成部门分组统计。
- pandas 基础 API：`pandas_1_read.py`、`pandas_2_select_filter.py`、`pandas_df_vs_series.py`、`pandas_manual_create.py`、`pandas_why_flexible.py`、`pandas_dict_list_connection.py` 展示 `read_csv`、`head/info/describe`、Series vs DataFrame、从 dict/list 构造以及双向转换。
- pandas 进阶与综合：`pandas_4_groupby.py`、`ex35_fund_data_analysis.py`、`ex36_pandas_data_cleaning.py` 覆盖 `groupby().agg()`、`transform` 广播、缺失值处理（`isna`/`fillna`/`dropna`）、去重、类型转换（`pd.to_numeric(errors='coerce')`）与 `to_csv` 导出。
所有脚本均为可独立运行的单文件程序，通过 `if __name__ == "__main__"` 或顶部注释说明任务目标与知识点提示，依赖外部 CSV 数据文件（如 `employees.csv`、`pandas_orders.csv`、`messy_employees.csv`），模块内部无相互 import。
