本模块为扁平的练习脚本集合，每个 exNN_*.py 是一个自包含的可执行单元：顶部以注释描述题目要求，随后定义若干纯函数（或类），并在 `if __name__ == "__main__":` 块中组织调用流程与打印输出；数据通常以内联变量提供，少数练习通过 `open()` 读写外部文件（如 employees.csv、books.json、student_scores.json）。依赖方向单一——各练习之间互不 import，仅使用 Python 标准库（json、csv、pprint）；ex19_student_class.py 是首个引入 class/__init__/self 的 OOP 示例，作为从过程式到面向对象过渡的里程碑。
