本模块由两个独立的可执行练习组成：
- ex27_file_io.py：单文件练习，定义 write_diary / append_diary / read_diary 三个函数分别对应 "w"（写入）、"a"（追加）、"r"（读取）三种 open 模式，统一使用 `with open(..., encoding="utf-8")` 上下文管理器操作 my_diary.txt，并通过 `if __name__ == "__main__":` 顺序串联三步。
- ex30_helper.py + ex30_main.py：双文件练习，ex30_helper.py 作为被导入模块暴露 greet() 与 add_numbers() 工具函数，并在其自身 `__main__` 块内做内部自测；ex30_main.py 通过 `import ex30_helper as hp` 以别名方式导入并调用这两个函数，同样用 `run_main()` + `__main__` 守卫组织入口。
依赖方向单向：ex30_main → ex30_helper，ex27 不依赖其他文件。
