本模块由 6 个相互独立的单文件脚本组成，每个文件对应一个编号练习，按难度递进组织：
- ex20/ex21：引入 `__str__`/`__repr__` 魔法方法与组合关系（Student + StudentManager）。
- ex22：用 Box 类演示 self 作为实例引用的语义。
- ex23：Book + BookManager，综合实现 `__eq__`/`__lt__`、property 校验、`__len__`/`__iter__`/`__contains__` 容器协议。
- ex24：在 ex21 基础上强化容器协议与基于 student_id 的相等规则，并引入 index 字典做 O(1) 查找。
- ex25：Employee 基类 + Manager/Developer 子类，演示继承与方法重写（多态）。
所有文件均为可独立运行的脚本，通过 `if __name__ == "__main__":` 块提供测试用例；文件之间无 import 依赖，仅共享同一组 OOP 概念。
