- 每个练习文件以中文注释开头说明题目目标，并在末尾用 `if __name__ == "__main__":` 块提供可直接运行的测试代码。
- 字符串表示统一实现 `__str__`，并通过 `__repr__ = __str__` 别名让列表打印也友好显示。
- 自定义比较方法（`__eq__`、`__lt__`）先使用 `isinstance(other, X)` 检查类型，不匹配时返回 `NotImplemented`。
- 数值型属性通过 `@property` 配合 setter 进行范围/类型校验，非法值抛出 `ValueError`。
- 容器管理器类（BookManager、StudentManager）同时实现 `__len__`、`__iter__`、`__contains__` 以支持 len()/for/in 语法。
