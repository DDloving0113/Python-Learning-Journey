### 本地记账本管理系统
- 定义：项目中的综合性 CLI 小项目（`project_expense_tracker.py`），基于面向对象设计，实现增删查改账单、统计总支出等功能，所有数据持久化为本地 JSON 文件。
- 别名：ExpenseTracker、expense tracker

### AI 交接卡片
- 定义：仓库根目录下的 `AI_交接卡片.md`，是面向 AI 助手的快速上下文文档，记录当前学习进度、已掌握知识点和关键文件路径，便于新会话快速承接学习状态。
- 别名：交接卡片

### Python 学习进度档案
- 定义：仓库根目录下 `Python_学习进度档案.md`，是一份约 358 行的详细能力追踪文档，按模块记录掌握程度与里程碑，是 AI 了解学习者真实水平的核心依据。
- 别名：学习进度档案

### 练习题 + 小项目 + 改作业式复盘
- 定义：学习者确认的学习方式：先做练习题巩固语法，再完成小项目整合知识，最后由 AI 对已完成代码进行批改、纠错与优化建议，而非直接代写答案。
- 别名：改作业式复盘

### go-live gate
- 定义：阶段成果完成后触发 git commit/push 的检查点，用于阶段性提交学习产出到版本库，形成可追溯的学习轨迹。
- 别名：git 提交检查点
