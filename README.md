# 🚀 Python Learning Journey

> 这是我从零开始学习 Python 的练习记录仓库。通过一步步的实战练习，我正在逐步掌握 Python 的核心语法、面向对象编程（OOP）以及实战项目的开发。

## 🎯 学习进度与里程碑

### 🌟 阶段一：基础语法通关 (Ex01 - Ex19)
- **数据类型**：掌握了 `int`, `float`, `str`, `bool` 等基础类型的操作。
- **数据结构**：熟练使用 `list`（列表）, `dict`（字典）, `set`（集合）进行数据存取与遍历。
- **控制流**：掌握了 `if/elif/else` 条件判断和 `for/while` 循环机制。
- **函数**：理解了函数的定义、返回值以及默认参数的使用。

### 🌟 阶段二：面向对象编程 (OOP) (Ex20 - Ex26)
- **类与对象**：深刻理解 `class`, `__init__`, 以及 `self` 的作用。
- **封装**：学会使用 `@property` 装饰器对属性进行安全校验与只读设置。
- **继承与多态**：通过员工工资系统项目，掌握了 `super()` 和方法重写。
- **高级特性**：理解并应用了**魔法方法**（如 `__str__`, `__len__`, `__eq__`）和**类方法**（`@classmethod`）。

### 🌟 阶段三：进阶与实战准备 (Ex27 - Ex30)
- **文件操作**：熟练使用 `with open()` 进行 `.txt` 的读写。
- **数据持久化**：掌握了利用 `json` 模块（`dump/load`）保存和读取结构化数据。
- **异常处理**：学会使用 `try...except...finally` 编写健壮的代码，防止程序崩溃。
- **模块化**：掌握了将代码拆分为多个文件并使用 `import as` 进行调用。

## 🛠️ 实战项目展示

### 💰 本地记账本管理系统 (Local Expense Tracker)
**文件**: `project_expense_tracker.py`

**项目简介**:
这是一个基于命令行的纯本地记账本小软件，将上述所有 Python 基础知识融会贯通。

**核心功能**:
1. **交互式菜单**：使用 `while True` 主循环构建了友好的 CLI 用户界面。
2. **面向对象设计**：使用 `ExpenseManager` 类封装了所有记账逻辑。
3. **数据持久化**：所有的账单都会自动保存为 `expenses.json` 文件，重启程序数据不丢失。
4. **防抖与容错**：加入了严格的类型转换校验 (`try/except ValueError`)，防止用户误输入导致崩溃。
5. **增删查改**：支持添加新账单、查看历史账单明细、统计总支出以及删除指定账单。
