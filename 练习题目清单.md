## 一、函数与返回值小测验

### 1）安全计算器函数

- 题目：  
  编写函数 `calc(a, b, op)`，支持四种运算：`+ - * /`  
  - 如果 `op` 是 `"+"`、`"-"`、`"*"`、`"/"` 之一，返回运算结果  
  - 如果 `op` 是其他字符串，返回 `"未知运算符"`  
  - 如果是除法且 `b` 为 `0`，返回 `"除数不能为 0"`  
  函数内部只用 `return` 返回结果，不在函数里 `print`。  
  再写一小段测试代码，调用这个函数，针对几组输入，把返回值 `print` 出来。

- 建议文件名：`ex01_calc_safe.py`

### 2）成绩等级评定（函数版）

- 题目：  
  参考你之前写过的成绩判断函数，现在要求拆成两个函数：
  1. `get_level(score)`  
     - 非数字或不在 `0–100` 范围内，返回 `"分数不合法"`  
     - 否则按区间返回 `"A" / "B" / "C" / "D"` 之一  
  2. `avg_level(cn, math, eng)`  
     - 先判断三个分数是否都合法（数字类型，且都在 `0–100` 之间）  
     - 有任何一个不合法时，返回 `"有不合法的分数"`  
     - 否则计算平均分，调用 `get_level` 得到等级，并返回 `(avg, level)` 这样的元组  
  主程序中调用 `avg_level`，把平均分和等级打印出来。

- 建议文件名：`ex02_grade_level.py`

### 3）质数工具函数

- 题目：  
  在你之前的质数判断基础上升级：
  1. 编写函数 `is_prime(n)`：  
     - 非整数或小于 `2`，返回 `False`  
     - 否则按常规方式判断是否为质数，返回 `True/False`
  2. 编写函数 `primes_in_range(a, b)`：  
     - 接受两个整数 `a, b`（允许 `a > b`）  
     - 返回 `[min(a, b), max(a, b)]` 区间内所有质数组成的列表  
  主程序中调用 `primes_in_range`，打印出返回的质数列表。

- 建议文件名：`ex03_prime_tools.py`

## 二、数据结构综合小测验

### 4）购物车结算升级版

- 题目：  
  给定购物车列表：

  ```python
  cart = [
      {"name": "牛奶", "price": 6, "count": 2, "discount": 0.9},
      {"name": "面包", "price": 4, "count": 3, "discount": 1.0},
      {"name": "鸡蛋", "price": 1, "count": 10, "discount": 0.8},
  ]
  ```

  要求：
  1. 写函数 `calc_item_total(item)`：返回单个商品的折后总价 `price * count * discount`  
  2. 写函数 `checkout(cart)`：  
     - 使用 `calc_item_total` 计算每个商品的折后总价  
     - 返回一个字典，例如：  
       `{"items": [{"name": "牛奶", "total": 10.8}, ...], "grand_total": 23.4}`  
  主程序中调用 `checkout`，把结果美观地打印出来。

- 建议文件名：`ex04_cart_checkout.py`

### 5）用户日志统计（进阶版）

- 题目：  
  给定原始日志列表：

  ```python
  logs = [
      "user1,login,2026-01-01",
      "user2,logout,2026-01-01",
      "user1,login,2026-01-02",
      "user1,view,2026-01-02",
      "user2,login,2026-01-02",
      "user3,login,2026-01-02",
  ]
  ```

  要求：
  1. 写函数 `clean_logs(raw_logs)`：  
     - 返回“清洗后的日志列表”，每条是形如  
       `{"user": "user1", "action": "login", "date": "2026-01-01"}` 的字典  
  2. 写函数 `stat_user_actions(clean_logs)`：  
     - 返回一个字典：`{"user1": 3, "user2": 2, ...}`，表示每个用户的操作次数  
  3. 写函数 `stat_action_counts(clean_logs)`：  
     - 返回一个字典：`{"login": 4, "logout": 1, "view": 1}`，表示每种操作的出现次数  
  主程序中依次调用两个统计函数，打印统计结果。

- 建议文件名：`ex05_log_stats.py`

### 6）学生成绩统计（升级版）

- 题目：  
  给定学生数据：

  ```python
  students = {
      "001": {"name": "小明", "score": {"语文": 85, "数学": 92}},
      "002": {"name": "小红", "score": {"语文": 90, "数学": 88}},
      "003": {"name": "小刚", "score": {"语文": 78, "数学": 80}},
  }
  ```

  要求：
  1. 写函数 `add_total_score(students)`：  
     - 在每个学生字典中增加一个 `total` 键，值为语文和数学总分  
  2. 写函数 `best_student(students)`：  
     - 返回总分最高的学生信息，例如：  
       `{"id": "002", "name": "小红", "total": 178}`  
  3. 写函数 `sorted_by_subject(students, subject)`：  
     - 按某个科目（"语文" 或 "数学"）分数从高到低排序  
     - 返回一个列表，例如：`[("小红", 90), ("小明", 85), ("小刚", 78)]`  
  主程序中测试：  
  - 打印总分最高的学生  
  - 打印按语文排序的列表  
  - 打印按数学排序的列表

- 建议文件名：`ex06_student_scores.py`

### 7）用户数据清洗与去重（强化版）

- 题目：  
  给定用户数据列表：

  ```python
  users = [
      {"name": "Alice", "age": 25},
      {"name": "Bob", "age": ""},
      {"name": "Alice", "age": 25},
      {"name": "Charlie", "age": 30},
      {"name": "", "age": 28},
  ]
  ```

  要求：
  1. 写函数 `clean_users(users)`：  
     - 过滤掉姓名为空或年龄为空的记录  
     - 返回有效用户列表  
  2. 写函数 `dedup_users(users)`：  
     - 把“姓名 + 年龄”完全相同的视为重复，只保留一条  
     - 返回去重后的列表  
  3. 写函数 `sort_users_by_age(users)`：  
     - 按年龄从小到大排序，返回排序后的列表  
  主程序中组合调用三个函数，输出最终“干净并按年龄排序”的用户列表。

- 建议文件名：`ex07_user_clean.py`
